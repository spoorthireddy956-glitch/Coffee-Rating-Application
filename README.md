# Coffee Rating Application

A simple web-based Coffee Rating Application built using **Flask (Python)** and **SQLite**.  
Users can view different coffee types and vote for their favorite ones. Each vote updates the count in real-time using backend logic.

---

## Features

- Display list of coffee items
-  Vote for your favorite coffee
- Real-time vote count update
-  Data stored using SQLite database
- Simple and clean UI using HTML & CSS

---

##  Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript (Fetch API)
